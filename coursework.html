Autumn 2018 CSE 1223 Introduction to Programming in Java
  • Learned basic programming techniques and concepts using Java (variables, types, computations, strings, boolean expressions, conditional statements, loops, methods, arrays, file I/O, object-oriented programming, using and implementing classes).
  • Introduced the Eclipse IDE as a tool for Java development.
  • Implemented simple programs, some of which from a provided skeleton, to demonstrate new concepts. Examples include:
    • A program prompting a user for a string, and displaying information about that string.
    • A game similar to "Rock, Paper, Scissors".
    • Implementing methods containing algorithms to convert between binary and decimal numbers, and write a main program that utilizes the algorithms implemented.
    • A program that writes a formatted table to a file, using data from an input file.
Spring 2019 CSE 2221 Software I: Software Components
  • Used Java as an introduction to Software Engineering concepts.
  • Reviewed/reintroduced Java and programming concepts.
    • Additionally introduced to parameter passing, references, and recursion.
  • Introduced to the design-by-contract principle.
  • Used an external library of software components to better understand their function and design.
  • Learned various concepts used in object-oriented programming (packages, interfaces, classes, inheritance, layering, object type vs declared type, object value vs reference value)
  • Introduced JUnit as a framework for unit testing in Java.
  • Implemented layered secondary methods using kernel methods and recursion.
  • Learned the basics on how a Graphical User Interface is implemented.
  • Programming assignments were slightly more complicated, and utilized the provided components. Examples include:
    • An RSS reader and aggregator using an XMLTree component, and displayed the output to a webpage using I/O components.
    • A utility class that provided methods to generate a reasonably large number and to check whether a number is prime, using a NaturalNumber component. JUnit tests were used to test the implementations of these methods.
    • A program that used various data structures (Queue, Set, etc) to read a glossary from an input file, and generate a webpage for each word in the glossary, linking to other words that may be inside the definition, with an index page listing all the words in alphebetical order.
  • For more information, see the course webpage here.
